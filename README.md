# OS-LAB
OS Lab – Java Implementations

This repository contains Operating Systems Lab programs implemented in Java, covering all major OS concepts such as CPU Scheduling, Disk Scheduling, Memory Allocation, Page Replacement, and Process Synchronization.

The project is structured module-wise for easy understanding and execution, following standard academic OS lab requirements.
